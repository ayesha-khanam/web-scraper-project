# web-scraper-project
This is a Python-based web scraper that collects headlines from news websites using requests and BeautifulSoup. It extracts the title, URL, and time if available, supports keyword filtering, multiple sources, and saves results in JSON or CSV format.

Features
Scrape headlines from news websites
Extract title, URL, and time
Support multiple websites
Keyword-based filtering
Save results to JSON or CSV
Delay between requests
Basic robots.txt checking
Error handling
